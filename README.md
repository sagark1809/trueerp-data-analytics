# TrueERP Data Analytics Project

## Overview
This project demonstrates data cleaning, data filtering, and data analysis
operations on ERP purchase bill data exported from the TrueERP system.

## Dataset
- All Purchase Bills.csv  
Contains supplier-wise purchase transactions including invoice details,
payment status, and tax deductions.

## Tools & Technologies
- Python
- Pandas
- Jupyter Notebook

## Tasks Performed
- Data loading and understanding
- Data cleaning (handling missing values, date conversion, standardization)
- Data filtering (outstanding bills, high-value purchases, supplier analysis)
- Data analysis (purchase trends, supplier dependency, payment efficiency, TDS)

## Key Insights
- Identified outstanding supplier liabilities
- Analyzed monthly procurement trends
- Evaluated supplier concentration
- Assessed payment efficiency and tax deductions

## Usage
Open the Jupyter Notebook file to view the complete step-by-step analysis.
